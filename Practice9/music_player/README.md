# Music Player

Interactive keyboard-controlled music player built with pygame.

## Keyboard Controls
| Key | Action |
|-----|--------|
| **P** | Play / Pause toggle |
| **S** | Stop |
| **N** | Next track |
| **B** | Previous (Back) track |
| **F** | Open folder picker to choose music folder |
| **Q** / **ESC** | Quit |

## Supported formats
MP3, WAV, OGG, FLAC

## Adding music
Drop audio files into the `music/` folder — they load automatically at startup.  
Press **F** at any time to reload a different folder.

## Requirements
```
pip install pygame
```

## Run
```bash
python main.py
```
